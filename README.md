
## 🛠️ วิธีการติดตั้ง

### 1. ติดตั้งเซิร์ฟเวอร์
- ใช้ XAMPP, WAMP, MAMP หรือ LAMP
- เปิด Apache และ MySQL

### 2. นำไฟล์ไปวางในโฟลเดอร์ `htdocs` (XAMPP) หรือเทียบเท่า
- ตั้งชื่อโฟลเดอร์โปรเจกต์ เช่น `gift-finder`

### 3. สร้างฐานข้อมูล
- เปิด phpMyAdmin
- สร้างฐานข้อมูลใหม่ชื่อ `gift_finder`
- นำไฟล์ `gift_finder.sql` มาอิมพอร์ต

### 4. ตั้งค่าเชื่อมต่อฐานข้อมูล
- แก้ไขไฟล์ `api/config.php`
- ใส่ข้อมูล username, password, host, database ให้ตรงกับเซิร์ฟเวอร์ของคุณ

### 5. เข้าถึงแอป
เปิดเบราว์เซอร์แล้วไปที่: http://localhost/

## ⚙️ วิธีการติดตั้งและรันโปรเจกต์

### 1. ติดตั้งเซิร์ฟเวอร์
- ใช้ XAMPP / WAMP / MAMP / LAMP
- เปิด Apache และ MySQL

### 2. วางโปรเจกต์
- วางโฟลเดอร์ `H3` ลงใน `htdocs` (XAMPP) หรือเทียบเท่า

- เปิด phpMyAdmin → สร้างฐานข้อมูลชื่อ `gift_finder`
- อิมพอร์ตไฟล์ `gift_finder.sql`

### 4. ตั้งค่าฐานข้อมูล
- แก้ไขไฟล์ `api/config.php`
- ใส่ข้อมูล:

  $username = "root"; // หรือชื่อผู้ใช้ของคุณ
